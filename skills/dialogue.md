# Dialogue Skill

Rules for every dialogue exchange:

1. Every line should ideally do two things simultaneously:
   - Advance the scene's conflict or information
   - Reveal character (voice, desire, fear)

2. Under pressure, people rarely speak in complete, tidy sentences.
   Short lines. Incomplete thoughts. Interruptions — where the genre calls for it.

3. Read every line aloud. If it sounds like it was written, consider rewriting it.

4. Prefer characters not explaining their own motivation directly.
   Weaker: "I'm doing this because I am afraid of losing you."
   Stronger: "Just — don't go yet. Please."

5. Subtext: the real conversation happening beneath the spoken one.

6. Voice differentiation: every named character in a project should be
   distinguishable from dialogue alone. Define each character's voice in
   their Character node (want/need/wound/flaw feed this).
