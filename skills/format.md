# Screenplay Format Skill

SLUGLINES: INT./EXT. LOCATION — TIME
Bold. All caps. No punctuation at end.
Example: EXT. WAREHOUSE ROOFTOP — NIGHT

ACTION LINES:
- Present tense always ("She walks" not "walked")
- One idea per paragraph — generally no more than 4 lines
- White space is free — use it
- Avoid camera directions (no "CLOSE ON", "PAN TO") in spec scripts
- Character names ALL CAPS on first introduction only

CHARACTER CUES: ALL CAPS, centred at ~3.5" from left margin

PARENTHETICALS: sparingly — only when delivery cannot be inferred

DIALOGUE: ~2.5" from left, ~2.5" from right margin

TRANSITIONS: SMASH CUT TO: / CUT TO: / FADE TO: — right-aligned, sparingly

PACING RULE: 1 page ≈ 1 minute of screen time
Target scene length: 2-4 pages, adjust per project/format (feature vs. TV vs. short)
